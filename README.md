Description:
Webpage for analyzing natural language sentiment. Uses Naive Bayes and the Bag-of-words model (implemented with Python) to classify text sentiment as "Positive" or "Negative".

Usage Instructions for Linux:

Step 1:
Install and start Docker if necessary. Open the terminal and enter `docker build -t myapp .` and `docker run -d -p 8080:8080 myapp`.

Step 2:
Open a new tab in any browser. Enter the url `http://localhost:8080/`. The web application should open.
